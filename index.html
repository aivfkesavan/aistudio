HaiVE AI Studio Installation Guide

Preliminary Step: Clear Old Installation

If you already have the app installed, open it and click the "Reset" button to clear all cache files. Remove the old application from your system. Proceed with the new installation.

Step 1: Download the Application

Click on the link to download the HaiVE AI Studio application:

  • For Mac: Download Link
  • For Windows: Windows Download Link (Note: Disable your antivirus software before installation to prevent any issues.)

Step 2: Handling the "Damaged" Error (Mac Only)

Attempt to open the application. If you receive a pop-up stating that "HaiVE AI Studio is damaged and can't be opened", follow the next steps.

Step 3: Removing Quarantine Attribute (Mac Only)

Open Terminal on your Mac. Run the following command to remove the quarantine attribute:

xattr -c /Applications/HaiVE\ AI\ Studio.app

Step 4: Opening the Application

Navigate to the Applications folder. Control-click the HaiVE AI Studio app. Select Open from the context menu and confirm by clicking Open again in the pop-up. Sign up or log in to your account to access the application.

Step 5: Downloading Models

Go to the Model Download Catalog screen. You will see a screen like the one below:

Review the status of each model to determine which one is suitable for your system. The status will indicate if a model is recommended, not suitable due to insufficient RAM, or might slow down your system. Click the Download button next to the model that suits your system specifications.

Step 6: Model Settings and Starting

Go to the Settings page. You will see a screen like the one below:

Choose the model you want to use from the drop-down menu. Adjust the model settings such as Context length, Continuous Batching, Temperature, Max Tokens, Repeat Penalty, Top P Sampling, and Token Limit. Monitor the Usage of CPU and RAM. Click the Start button to start the model.

Step 7: Chatting with the System

Go to the Chat screen. You will see a screen like the one below:

Start a new chat by clicking the "Add New Chat" button. Interact with the AI by typing your messages in the message box. You can see the chat history and manage your conversations.

Step 8: Sharing Local Inference

Go to the Local Inference screen. You will see a screen like the one below:

Click the "Go Public" button to share your local inference. Invite friends by entering their email addresses and clicking the "Invite" button. After successful sharing, you will see a red "Stop Sharing" button.

Step 9: Accessing Inference on Mobile

Download the mobile app:

  • For Android users, Download the APK
  • For Apple users, Download from the App Store

Sign up or log in to your account if you don't already have one. Set up the server. If your local inference is on, you will see the option to connect:

Click Connect and proceed to the chat screen to start chatting.
